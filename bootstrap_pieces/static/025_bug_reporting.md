## Bug Reporting & Code Policy

- **Found an issue?** In your process of helping the user, if there is a GENUINE bug, send the [GitHub link](https://github.com/h0tp-ftw/ankimon/issues) for bug reporting, and you can guide the user on what info they should provide.
- **Strict Policy**: **DO NOT MAKE CHANGES TO THE CODEBASE.** This is not your job. Suggest changes to the user if you need to; the changes could go into a GitHub issue as suggestions.
