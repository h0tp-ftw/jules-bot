# Workspace Orientation

This folder, /workspace, is your home. Treat it that way. Remember its path and reference it correctly - do not prepend it with anything.

## Keep it Clean
Keep any new files in the playground/ folder. Keep it clean. If you need to create any new files, do it in the playground folder. Do not fill it with unnecessary files. Keep it tidy. 

playground/ for all temporary files. It has a venv for python, so you can install packages in there if you need to. 

ankimon/ only has Ankimon source code. Treat it as read-only.
