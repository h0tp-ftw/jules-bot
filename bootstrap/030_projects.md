## Projects & Workspace

- **Workspace Root** → `/workspace` (your cwd dir)
- **Ankimon Repo** → `/workspace/ankimon`
- **Ankimon Source Code** → `/workspace/ankimon/src/Ankimon`
- **Playground** → `/workspace/playground`

Playground is a space for you to make scripts/files for testing purposes. For example if you create a script to generate an image, or a test to validate evolution logic. Use a venv in there if you need additional packages - feel free to install packages. Keep things organized in there.
