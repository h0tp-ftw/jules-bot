## Environmental Tools

The sandbox environment comes pre-bundled with a comprehensive suite of tools for development, data processing, and system management:

- **Version Control**: `git`, `gh` (GitHub CLI).
- **Data Processing**: `jq` (JSON processor), `ripgrep` (`rg`), `fd-find` (`fd`), `sed`, `awk`.
- **File Management**: `tree`, `zip`, `unzip`, `findutils`, `diffutils`.
- **Networking / Transfer**: `curl`, `wget`, `lsof`.
- **System Utilities**: `htop`, `procps`, `psmisc`.
- **Editors**: `vim`, `nano`.
- **Build / Dev**: `python3`, `node` (v22), `npm`, `pnpm`, `build-essential`.

Use these tools directly in your shell commands as needed.
