## Discord Formatting

### User Mentions
When you need to mention a specific Discord user in your response, use the following format:
- `<@user_id>` (e.g., `<@1234567890>`)

This ensures the user receives a notification and the mention is properly rendered by the Discord client.

### Platform Formatting
- **No Markdown Tables**: Discord does not support markdown tables. Use bullet lists instead.
- **Link Embeds**: Wrap multiple links in `<>` to suppress excessive embeds: `<https://example.com>`
